# Metodoligia-Benjamim-Graham

# Projeto: Pilares do Investimento de Valor com Benjamin Graham

Este repositório contém um caderno temático desenvolvido no **NotebookLM** como parte de um desafio prático da DIO. O objetivo é explorar a metodologia de Benjamin Graham, o "pai do Value Investing", utilizando Inteligência Artificial para curadoria e organização do conhecimento.

---

## 🎯 Contexto e Objetivos

O foco deste estudo é a arquitetura conceitual de Benjamin Graham, mentor de Warren Buffett. Os principais objetivos foram:
*   Diferenciar **Investimento de Especulação**.
*   Compreender a aplicação prática da **Margem de Segurança**.
*   Sistematizar os critérios de seleção de ações para o **Investidor Defensivo**.

---

## 📚 Curadoria de Fontes

Para alimentar a base de conhecimento no NotebookLM, foram utilizadas as seguintes fontes:
1.  **O Investidor Inteligente**: Transcrições e áudios sobre a psicologia do investidor.
2.  **Metodologia de Investimento de Valor**: Artigo técnico sobre modelagem quantitativa.
3.  **Fórmula de Graham**: Guia prático de cálculo de valor intrínseco.
4.  **Ten Stock Selection Criteria (Moomoo/Investing.com)**: Critérios quantitativos de triagem.

---

## 🛠️ Engenharia de Prompts e "Cicatrizes"

Durante a interação com a IA, documentei os seguintes aprendizados:

*   **Prompt Exploratório:** *"Quais são os principais pilares da metodologia de Benjamin Graham?"*
    *   *Resultado:* A IA trouxe conceitos densos; precisei refinar para tópicos.
*   **Prompt de Refinamento:** *"Liste em tópicos os critérios quantitativos para o investidor defensivo."*
    *   *Desafio (Cicatriz):* A IA misturou inicialmente critérios de crescimento com valor. Foi necessário pedir a separação explícita entre investidores defensivos e empreendedores.
