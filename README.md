# Metodoligia-Benjamim-Graham

Contexto e Objetivos
Este projeto utiliza a IA NotebookLM para sistematizar os pilares do Value Investing (Investimento de Valor) de Benjamin Graham
. O objetivo é criar uma base de conhecimento para investidores iniciantes (defensivos) e ativos (empreendedores) baseada na análise fundamentalista e disciplina emocional
.
Curadoria de Fontes
As principais fontes utilizadas foram:
O Investidor Inteligente: A base teórica sobre a psicologia do investidor e o Sr. Mercado
.
Fórmula de Graham: Metodologia quantitativa para encontrar o valor intrínseco
.
Critérios de Seleção (Moomoo/Investing.com): Regras práticas para triagem de ações
.
Engenharia de Prompts e "Cicatrizes"
Documente como refinamos a informação:
Prompt Exploratório: "Quais são os principais pilares da metodologia de Benjamin Graham?"
Aprendizado: A resposta inicial focou muito na teoria; precisei pedir uma lista em tópicos para organizar conceitos como Margem de Segurança e Sr. Mercado [Conversa anterior].
Prompt de Refinamento: "Liste em tópicos os critérios para o investidor defensivo."
Aprendizado: Identificamos a necessidade de separar o investidor passivo (defensivo) do ativo (empreendedor), focando em critérios como P/L < 15 e P/VP < 1,5
.
